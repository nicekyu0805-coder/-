# YouTube DeepSearch 🔍

YouTube 동영상을 심층적으로 검색하고 분석할 수 있는 고급 검색 도구입니다. YouTube Data API v3를 활용하여 다양한 필터와 정렬 옵션을 제공하며, 검색 결과를 효과적으로 관리할 수 있습니다.

## ✨ 주요 기능

### 🔍 고급 검색 기능
- **키워드 검색**: 제목, 설명, 태그를 포함한 종합적인 검색
- **고급 필터**: 동영상 길이, 업로드 날짜, 정렬 순서, 결과 개수 설정
- **조회수 범위 필터**: 1천~1천만 이상까지 9단계 세분화된 조회수 필터링
- **구독자수 범위 필터**: 1천~500만 이상까지 11단계 채널 규모별 필터링
- **실시간 검색 제안**: 검색어 입력 시 자동완성 기능
- **스마트 필터링**: 클라이언트 측에서 추가적인 정밀 필터링 제공

### 📊 검색 결과 관리
- **다양한 보기 모드**: 그리드 뷰와 리스트 뷰 지원
- **상세 정보 표시**: 조회수, 좋아요 수, 구독자수, 게시일, 채널 정보
- **결과 내보내기**: CSV, JSON 형식으로 검색 결과 다운로드
- **페이지네이션**: 대량의 검색 결과를 효율적으로 탐색
- **필터 상태 표시**: 적용된 필터 조건을 결과 헤더에 명시

### ❤️ 개인화 기능
- **즐겨찾기**: 마음에 드는 동영상을 저장하고 관리
- **검색 기록**: 이전 검색어와 필터 설정을 저장하고 재사용
- **설정 저장**: 사용자 선호도에 맞는 기본 설정 유지

### 🎨 사용자 경험
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화
- **직관적인 UI**: 깔끔하고 현대적인 인터페이스
- **키보드 단축키**: Ctrl+K로 검색 포커스, ESC로 모달 닫기
- **접근성**: 스크린 리더와 키보드 네비게이션 지원

## 🚀 사용 방법

### 1. API 키 설정
1. [Google Cloud Console](https://console.developers.google.com)에 접속
2. 새 프로젝트를 생성하거나 기존 프로젝트 선택
3. YouTube Data API v3를 활성화
4. API 키를 생성 (웹 애플리케이션용)
5. 웹사이트 첫 방문 시 나타나는 모달에서 API 키 입력

### 2. 기본 검색
1. 검색어를 입력창에 입력
2. "검색" 버튼 클릭 또는 Enter 키 누르기
3. 결과를 그리드 또는 리스트 형태로 확인

### 3. 고급 검색
1. "고급 필터" 버튼 클릭하여 필터 옵션 표시
2. 원하는 필터 설정:
   - **동영상 길이**: 4분 미만, 4-20분, 20분 초과
   - **업로드 날짜**: 지난 1시간, 오늘, 이번 주, 이번 달, 올해
   - **조회수 범위**: 1천~1천만 이상, 구간별 세분화 필터링
   - **구독자수 범위**: 1천~500만 이상, 채널 규모별 필터링
   - **정렬 순서**: 관련성, 최신순, 조회수, 평점
   - **결과 개수**: 10개, 25개, 50개
3. 검색 실행

### 4. 결과 관리
- **동영상 재생**: 썸네일 또는 "재생" 버튼 클릭
- **상세 정보**: "상세정보" 버튼으로 동영상 메타데이터 확인
- **즐겨찾기**: 하트 버튼으로 즐겨찾기 추가/제거
- **결과 내보내기**: "내보내기" 버튼으로 CSV 파일 다운로드

## 🔧 설정 옵션

### API 설정
```javascript
// js/config.js에서 수정 가능
YOUTUBE_API: {
    KEY: 'YOUR_API_KEY_HERE',
    MAX_RESULTS_PER_REQUEST: 50,
    DEFAULT_RESULTS: 25,
    REGION_CODE: 'KR',
    LANGUAGE: 'ko'
}
```

### 애플리케이션 설정
```javascript
APP: {
    MAX_HISTORY_ITEMS: 100,    // 최대 검색 기록 개수
    MAX_FAVORITES: 500,        // 최대 즐겨찾기 개수
    DEBUG: false               // 디버그 모드
}
```

## 🔐 보안 고려사항

### API 키 보호
- API 키는 클라이언트 측에 저장되므로 공개될 수 있음
- Google Cloud Console에서 도메인 제한 설정 권장
- API 키 사용량 모니터링 및 제한 설정

### 데이터 보안
- 모든 사용자 데이터는 로컬 브라우저에만 저장
- 외부 서버로 개인정보 전송하지 않음
- HTTPS 사용 권장

## 🚧 알려진 제한사항

1. **YouTube API 할당량**: 일일 API 호출 제한이 있음
2. **브라우저 스토리지**: 로컬 스토리지 용량 제한 (일반적으로 5-10MB)
3. **CORS 제한**: 일부 기능은 HTTPS 환경에서만 동작
4. **모바일 성능**: 대량의 검색 결과 시 성능 저하 가능
